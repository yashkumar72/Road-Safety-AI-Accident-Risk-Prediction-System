# 🚦 Road Safety AI – Accident Risk Prediction System

An **AI-powered Full Stack Web Application** designed to analyze driving conditions and predict the **risk level of road accidents**.
The system helps improve road safety awareness by identifying **high-risk situations** based on factors such as location, time, weather, and traffic patterns.

---

## 📌 Project Overview

Road accidents are a major global concern. This project uses **Artificial Intelligence and Data Analysis** to predict accident risks and provide safety awareness.

Users can input driving conditions like **state, city, area, time, day/night, and rain conditions**, and the system predicts whether the risk level is:

* 🟢 Low Risk
* 🟠 Medium Risk
* 🔴 High Risk

The project also includes **safety tips, accident prevention guidelines, statistical dashboards, and high-risk zone identification**.

---

## 🎯 Project Objective

The main objective of this project is to:

* Predict **high-risk accident situations** using AI.
* Improve **road safety awareness**.
* Provide **data visualization and insights** using dashboards.
* Help drivers understand **when and where accidents are more likely to occur**.

---

## ⚙️ How the System Works

1. Users enter driving conditions:

   * State
   * City
   * Area
   * Time
   * Day or Night
   * Rain Condition

2. The system sends the data to the **Flask backend**.

3. The **Machine Learning model** processes the input.

4. The system predicts the **accident risk level**:

   * Low
   * Medium
   * High
