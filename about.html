About of this Project

AI Based Road Safety and Accident Prediction System

This project is a full stack web application designed to improve road safety using Artificial Intelligence. The system analyzes different driving conditions such as location, time, weather, and road situations to predict the risk level of road accidents.

Project Objective

The main objective of this project is to reduce road accidents by predicting high-risk situations and providing awareness about road safety. By analyzing accident patterns, the system helps drivers understand when and where accidents are more likely to occur.

How the System Works

Users enter information such as state, city, area, time, day or night, and rain conditions. The system processes this data using an AI prediction model implemented in Python and Flask. Based on these inputs, the system predicts the accident risk level as Low, Medium, or High.

Technologies Used

Frontend: HTML, CSS, JavaScript
Backend: Python Flask
AI / ML: Python, Scikit-learn, Machine Learning Model, Accident Risk Classification (Low / Medium / High)
Data Analysis: Pandas, NumPy, Dataset Processing, Data Preprocessing, Feature Analysis
Visualization: Chart.js
Dataset: Custom Road Safety Dataset

Features of the System

  • Accident Risk Prediction using AI
  • Road Safety Awareness Tips
  • Accident Prevention Guidelines
  • Interactive Dashboard Visualization
  • High Risk Zone Identification
